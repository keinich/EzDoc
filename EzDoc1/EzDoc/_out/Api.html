Bedingung

Beschreibt eine Bedingung unter der ein Konto in einer zu einem gegebenen Berechnungskennzeichen gehörenden Kontofortschreibung relevant ist.

Methods

Name Beschreibung

Kontostatusbedingt Kontostatusbedingung, die nur die übergebenen Kontostatus zulässt.
Custom Erzeugt eine Custombedingung und führt zum Aufruf von IKontofortschreibungCustomizingProvider.IstKontoRelevantCustom.
