StandardRatenberechnung

Ratenberechnungskern. Kontostand wird gemäß Verzinsungsinformationen verzinst bis zum ersten Auszahlungszeitpunkt. Die ersta Rate ist der verzinste Kontostand geteilt durch die Anzahl der Raten. Jede weitere Rate wird durch Verzinsung gemäß Parameter berechnet.

Hibliberechnungsart

Art der Hibliberechnung. HibliLeistungAusVerstorbenenleistungBerechnen bedeutet, dass zuerst Leistungen des Verstorbenen ermittelt werden, aus denen dann die Hiblileistungen berechnet werden. HibliLeistungDirektBerechnen bedeutet, dass ohne Zuhilfenahme von Leistungen des Verstorbenen die Hiblileistungen direkt aus dem Kontostand berechnet werden.
