BeitragsberechnungMitEinkommensprozente

Berechnet einen Jahresbeitrag. Es werden zunächst Entgeltinformation und Dienstzeitinformation im Zeitraum vom letzten Jahrestag vor Berechnungsstichtag gemäß EinkommenJahrestag bis zum Berechnungsstichtag berechnet. Aus diesen Informationen wird mit BerechneBeiträgeMitEinkommensprozenten ein Beitrag berechnet.

Properties

Company Company
InstantieteBeitragMethod Falls die berechneten Beiträge nicht vom Type PensionLib.Domain.Beitrag sein sollen, kann in dieser Funktion ein anderer Typ instanziiert werden.
AddOrUpdateBeitragMethod Falls nötig, kann hier die AddOrUpdate-Funktion auf dem konkreten Beitragstyp aufgerufen werden, falls der PensionLib.Domain.Beitrag so erweitert wurde, dass sich auch die Eindeutigkeit verändert hat. Falls True zurückgegeben wird, findet kein Mapping vom Framework statt.
